# 가상 물리 실험실 (physics-lab)

코딩더메이커 수업용 물리 실험 시뮬레이터.
레고 스키 미니피겨 경사면 실험과 갈릴레이 자유낙하가 **"질량은 결과를 바꾸지 못한다"**는
같은 원리임을 확인하는 수업 도구다.

- 실험 1 · 경사면 활주 — 질량 A/B, 경사각, 마찰계수, 공기저항 조절
- 실험 2 · 갈릴레이 자유낙하 — 깃털 vs 쇠공, 진공/공기 토글
- 실시간 v-t 그래프, 실험 기록 테이블(최근 8회), 판정 배지
- 외부 라이브러리 없음, 단일 `index.html`, 태블릿 동작

접속 주소 ──────────────────────────────── https://by2k2020.github.io/physics-lab/
