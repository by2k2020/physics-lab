📋 조건별 기록표

🎪 소녀는 어떻게 외줄 위에서 안 떨어질까?

비밀은 무게중심이야. 무게중심이 줄 바로 위에 있으면 균형, 옆으로 벗어나면 그쪽으로 넘어져!

⚖ 무게중심이 뭐야?

물체의 무게가 한 점에 다 모여 있다고 생각할 수 있는 점이야. 시뮬레이터의 노란 점이 바로 소녀+봉+추 전체의 무게중심이야. 이 점이 줄에서 멀어질수록 위험해.

🤝 균형 힘 vs 불균형 힘

왼쪽 추와 오른쪽 추가 똑같으면 양쪽이 잡아당기는 힘이 비겨서(균형) 가만히 있어. 한쪽만 무거우면 힘이 불균형해져서 무거운 쪽으로 도는 힘이 생겨 — 그래서 넘어지는 거야.

🥢 균형봉은 왜 길까?

긴 봉은 돌리기 어려워. 그래서 바람이 밀어도 천천히 기울어지고, 소녀가 다시 잡을 시간이 생겨. 짧은 봉은 휙휙 빨리 넘어가서 잡을 틈이 없어!

⬇ 추를 아래로 늘어뜨리는 이유

추가 줄보다 아래에 매달리면 전체 무게중심이 내려가. 무게중심이 낮을수록 오뚝이처럼 잘 안 넘어져. 줄타기 선수의 봉이 아래로 휘어 있는 이유야.

💪 돌리는 힘(모멘트) = 무게 × 거리

추 1개라도 긴 봉 끝에 걸면 더 세게 돌아가! 돌리는 힘은 무게 × 줄에서 떨어진 거리거든. 그래서 긴 봉에 한쪽만 추를 걸면 짧은 봉보다 더 빨리 넘어져. 직접 비교해 봐.

🧑‍🏫 수업 이렇게 진행하세요 (5단계)

  • ① 맨몸 도전 — 추 0개·바람 0으로 시작. "가만히 있어도 조금씩 흔들리네?" (10초 성공 쉬움)
  • ② 불균형 실험 — 예측 모드를 켜고 왼쪽에만 추 2개. "어떻게 될까?" 손들고 → 실행 → 왼쪽으로 낙하 확인
  • ③ 균형 회복 — 오른쪽에도 추 2개를 걸어 대칭으로. "다시 균형!" — 균형/불균형 힘 정리
  • ④ 바람 도전 — 바람 3 + 짧은 봉 → 금방 낙하. 긴 봉 + 추 3:3으로 바꾸면? → 버틸 수 있다!
  • ⑤ 기록 경쟁 — 조별로 "바람 3에서 10초 버티는 설정"을 찾고 기록표 비교
🧱 레고 실험 팁
· 실물 브릭큐 모델도 똑같이 양쪽 무게가 같아야 소녀가 서 있어 — 시뮬로 예측하고 실물 추를 옮겨 확인
· 확장미션: 바람 3에서 10초 버티는 조합은 몇 개나 될까? 기록표에 모아 보기
· 순서는 항상: 시뮬로 예측 → 레고로 실험 → "왜 같을까/다를까?" 이야기
