스틱으로 퍽을 탁! 치는 게 바로 미는 힘이야.
세게 밀수록 퍽은 빨라지고, 더 멀리 가!
그런데 퍽은 왜 점점 느려질까? 바닥이 퍽을 붙잡는 힘, 마찰 때문이야.
🧊 빙판은 미끌미끌 → 마찰이 작아서 쭉~ 미끄러져.
🟫 거친 바닥은 까끌까끌 → 마찰이 커서 금방 멈춰 버려.
⭐ 그래서 규칙은 이거야!
힘이 셀수록, 마찰이 작을수록 → 퍽은 더 멀리 간다!
거친 바닥에서 골을 넣고 싶으면? 훨씬 더 세게 쳐야 해.
🧤 골키퍼는 좌우로 왔다갔다 해. 키퍼가 반대쪽으로 갈 때를 노려서 빈 곳으로 슛! — 힘 조절 + 방향 조절 + 타이밍, 세 가지가 다 필요해.

🧑‍🏫 수업 이렇게 진행하세요 (5단계)

  • ① 예측 — "예측 모드"를 켜고, 슛 전에 "이 세기면 골대까지 갈까?" 손들기
  • ② 슛! — 퍽이 미끄러지다 느려지는 모습 관찰. 예측과 비교
  • ③ 바닥 바꾸기 — 같은 세기로 빙판 → 거친 바닥. "왜 금방 멈추지?" (마찰!)
  • ④ 자동 실험 — 세기 1~8 거리를 표로 채우고, 몇부터 골대에 닿는지 찾기
  • ⑤ 골키퍼 대결 — 골키퍼를 켜고, 타이밍·방향을 읽어 3골 넣기 미션
🧱 레고 실험 팁
· 실물 BricQ 하키 모델도 똑같다 — 고무줄을 많이 감을수록(=세기↑) 퍽이 멀리 간다
· 책상 위(매끈)와 카펫 위(거침)에서 같은 세기로 쏘아 거리 비교 — 시뮬 표와 맞춰보기
· 순서는 항상: 시뮬로 예측 → 레고로 실험 → "왜 다르지?" 이야기
🔎 공식이 궁금한 선생님만 (수업에선 안 써도 됩니다)
마찰 감속: a = μg — 퍽 속도가 일정하게 줄어듭니다 (그래프가 직선인 이유).
도달 거리: d = v₀² / (2μg) — 속도의 제곱에 비례해, 세기 2배면 거리는 4배.
시뮬 값: 빙판 μ=0.05, 거친 바닥 μ=0.20 (4배) → 같은 세기의 거리도 ¼로 줄어듭니다.
